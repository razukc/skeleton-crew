# Example 03: Action Engine

**Core Feature**: Action registration and execution

This example demonstrates the Action Engine in isolation.

## What You'll Learn

- How to register actions with handlers
- Executing actions with and without parameters
- Action return values
- Listing all registered actions

## Run It

```bash
npm run example:03
```

## Key Concepts

- Actions are executable operations with handlers
- Actions can accept typed parameters
- Actions can return values (sync or async)
- Action handlers contain the business logic
