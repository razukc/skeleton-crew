# Example 04: Event Bus

**Core Feature**: Event emission and subscription

This example demonstrates the Event Bus for cross-plugin communication.

## What You'll Learn

- How to emit events from plugins
- How to subscribe to events
- Multiple subscribers for the same event
- Event-driven plugin communication

## Run It

```bash
npm run example:04
```

## Key Concepts

- Events enable loose coupling between plugins
- Multiple plugins can subscribe to the same event
- Events can carry data payloads
- Plugins don't need direct references to each other
