# Example 02: Screen Registry

**Core Feature**: Screen registration and retrieval

This example focuses solely on the Screen Registry subsystem.

## What You'll Learn

- How to register screens in a plugin
- Screen properties (id, title, component, metadata)
- Retrieving screens by ID
- Listing all registered screens

## Run It

```bash
npm run example:02
```

## Key Concepts

- Screens are declarative UI definitions
- Each screen has a unique ID
- Screens can include custom metadata
- Screen registry provides lookup and listing methods
